# ERC721 NFT Reference

> Contract: `StandardERC721.sol` — OZ ERC721 + URIStorage + Pausable + Ownable.

## deploy

### Command Template

```bash
DEPLOYER=$(cast wallet address --private-key $PRIVATE_KEY)

export NFT_NAME="<your-collection-name>"        # e.g. "Acme Art"
export NFT_SYMBOL="<YOUR_SYMBOL>"               # e.g. "ACM"
export NFT_BASE_URI="<your-metadata-base-uri>"  # e.g. "ipfs://Qm.../"; per-token URI set at mint

forge script contracts/script/Deploy.s.sol:DeployERC721 \
  --rpc-url $RPC_URL \
  --private-key $PRIVATE_KEY \
  -vvvv

forge script contracts/script/Deploy.s.sol:DeployERC721 \
  --rpc-url $RPC_URL \
  --private-key $PRIVATE_KEY \
  --broadcast \
  -vvvv
```

### Parameters

| Param | Description |
|-------|-------------|
| `NFT_NAME` | Collection name |
| `NFT_SYMBOL` | Collection symbol |
| `NFT_BASE_URI` | Base URI for metadata (append tokenId or per-token URI override) |

### Output Parsing

Same as ERC20 deploy. Hand off to `pharos-contract-verify`.

Constructor args for verification:
```bash
cast abi-encode "constructor(string,string,string,address)" \
  "$NFT_NAME" "$NFT_SYMBOL" "$NFT_BASE_URI" $DEPLOYER
```

> **Agent Guidelines:**
> 1. Complete Write Operation Pre-checks (see _guardrails.md).
> 2. Recommend pinning metadata to IPFS for permanence.
> 3. Recommend Safe ownership.

## mint

### Command Template

```bash
# Mint single NFT with custom URI
cast send $NFT_ADDR \
  "safeMint(address,string)" $RECIPIENT "ipfs://Qm.../0.json" \
  --rpc-url $RPC_URL \
  --private-key $PRIVATE_KEY
```

### Output Parsing

```
transactionHash: 0xTXHASH
```
Confirm: `cast call $NFT_ADDR "ownerOf(uint256)" 0 --rpc-url $RPC_URL`

Token IDs increment from 0 automatically.

### Error Handling

| Error | Cause | Fix |
|-------|-------|-----|
| `OwnableUnauthorizedAccount` | Not owner | Use owner key |
| `ERC721InvalidReceiver` | Recipient contract doesn't implement IERC721Receiver | Use EOA or deploy receiver contract |
| `EnforcedPause` | Collection paused | Unpause first |

## read-ops

```bash
# Owner of token
cast call $NFT_ADDR "ownerOf(uint256)" $TOKEN_ID --rpc-url $RPC_URL

# Token URI
cast call $NFT_ADDR "tokenURI(uint256)" $TOKEN_ID --rpc-url $RPC_URL | cast --to-ascii

# Balance
cast call $NFT_ADDR "balanceOf(address)" $HOLDER --rpc-url $RPC_URL | cast to-dec

# Collection owner
cast call $NFT_ADDR "owner()" --rpc-url $RPC_URL

# Paused
cast call $NFT_ADDR "paused()" --rpc-url $RPC_URL
```
