# ERC1155 Multi-Token Reference

> Contract: `StandardERC1155.sol` — OZ ERC1155 + Pausable + Supply + Ownable.

## deploy

### Command Template

```bash
DEPLOYER=$(cast wallet address --private-key $PRIVATE_KEY)

export MULTI_NAME="<your-collection-name>"          # e.g. "Acme Items"
export MULTI_SYMBOL="<YOUR_SYMBOL>"                # e.g. "ACMI"
export MULTI_URI="<your-metadata-uri-template>"    # e.g. "ipfs://Qm.../{id}.json"; {id} replaced with hex token ID

forge script contracts/script/Deploy.s.sol:DeployERC1155 \
  --rpc-url $RPC_URL \
  --private-key $PRIVATE_KEY \
  --broadcast \
  -vvvv
```

### Parameters

| Param | Description |
|-------|-------------|
| `MULTI_NAME` | Collection name |
| `MULTI_SYMBOL` | Symbol |
| `MULTI_URI` | Metadata URI template. `{id}` replaced by token ID in hex |

Constructor args for verification:
```bash
cast abi-encode "constructor(string,string,string,address)" \
  "$MULTI_NAME" "$MULTI_SYMBOL" "$MULTI_URI" $DEPLOYER
```

## mint

### Command Template

```bash
# Single mint
cast send $MULTI_ADDR \
  "mint(address,uint256,uint256,bytes)" $RECIPIENT $TOKEN_ID $AMOUNT "0x" \
  --rpc-url $RPC_URL \
  --private-key $PRIVATE_KEY

# Batch mint
cast send $MULTI_ADDR \
  "mintBatch(address,uint256[],uint256[],bytes)" \
  $RECIPIENT "[1,2,3]" "[100,200,50]" "0x" \
  --rpc-url $RPC_URL \
  --private-key $PRIVATE_KEY
```

### Error Handling

| Error | Cause | Fix |
|-------|-------|-----|
| `OwnableUnauthorizedAccount` | Not owner | Use owner key |
| `ERC1155InvalidArrayLength` | ids.length != amounts.length | Match array sizes |
| `EnforcedPause` | Paused | Unpause first |

## read-ops

```bash
# Balance of token ID
cast call $MULTI_ADDR "balanceOf(address,uint256)" $HOLDER $TOKEN_ID --rpc-url $RPC_URL | cast to-dec

# Total supply of token ID
cast call $MULTI_ADDR "totalSupply(uint256)" $TOKEN_ID --rpc-url $RPC_URL | cast to-dec

# URI
cast call $MULTI_ADDR "uri(uint256)" $TOKEN_ID --rpc-url $RPC_URL | cast --to-ascii

# Owner
cast call $MULTI_ADDR "owner()" --rpc-url $RPC_URL
```
